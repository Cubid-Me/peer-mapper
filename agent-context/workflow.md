# workflow.md

This file contains the mandatory per-session checklist agents must follow.

1. Read `AGENTS.md`
2. If you have specific session objectives, execute them. else read `agent-instructions.md` and execute the next session objectives as described therein.
3. Make safe changes and add tests.
4. Run `pnpm env:sync` and `pnpm spec:lint` if present.
5. Update `technical-spec.md` and `functional-spec.md` and other relevant docs as work proceeds.
6. Update `session-log.md` (append new session at top and bump session version).
